# App - JGO

En este repositorio se tiene el desarrollo de la aplicacion de pedidos.

## Tech Stack

- Angular
- Ionic
- Capacitor
- Firebase

## Features

- Login
- Registro
- Restablecer contraseña
- Gestiòn de productos
- Gestiòn de pedidos
- Gestiòn de perfil
- Carrito
